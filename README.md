
# Take Home Challenge


## Available Scripts

In the project directory, you can run:
client side :

### `cd Take-home_React`
### `cd challenge_react`
### `npm install`
### `npm start`

Server side:

### `cd Take_home_challenge`
### `cd TakeHome`
### `python manage.py runserver`
### `python manage.py makemigrations`
### `python manage.py makemigrate`


Access the application in your browser at : 
### `http://localhost:3000/`
